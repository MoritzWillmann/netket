run(n_iter, out=None, obs=None, show_progress=True, save_params_every=50, write_every=50, step_size=1, callback=<function AbstractVariationalDriver.<lambda>>)ÔÉÅ

Executes the Monte Carlo Variational optimization, updating the weights of the network stored in this driver for n_iter steps and dumping values of the observables obs in the output logger. If no logger is specified, creates a json file at out, overwriting files with the same prefix.

By default uses netket.logging.JsonLog. To know about the output format check it‚Äôs documentation. The logger object is also returned at the end of this function so that you can inspect the results without reading the json output.

Parameters

  • n_iter ‚Äì the total number of iterations

  • out ‚Äì A logger object, or an iterable of loggers, to be used to store simulation log and data. If this argument is a string, it will be used as output prefix for the standard JSON logger.

  • obs ‚Äì An iterable containing all observables that should be computed

  • save_params_every ‚Äì Every how many steps the parameters of the network should be serialized to disk (ignored if logger is provided)

  • write_every ‚Äì Every how many steps the json data should be flushed to disk (ignored if logger is provided)

  • step_size ‚Äì Every how many steps should observables be logged to disk (default=1)

  • show_progress ‚Äì If true displays a progress bar (default=True)

  • callback ‚Äì Callable or list of callable callback functions to stop training given a condition
